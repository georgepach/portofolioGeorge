Project Title

DataTani

Year

Apr 2025 - Present

Role/Position

Back-End & Front-End Developer

Project Description

DataTani is a mobile application developed in collaboration with the Forestry Department of Lampung to support farmers in estimating rice yields per hectare. I worked on both the frontend and backend, designing the interface and implementing the core calculation features. This provided a practical digital solution for farmers to calculate harvest outputs more efficiently. Through this experience, I strengthened my skills in full-stack mobile development and learned how technology can be applied to address challenges in agriculture.
