Project Description

GreenLoop is a group project from a Software Engineering course at Bina Nusantara University, developed as a web-based platform to support waste exchange and collection. Each team member contributed to UI/UX design, frontend, and backend development, while I took the role of project manager. We organised daily scrums, facilitated discussions on project phases, and ensured coordination across design and implementation. Through this project we created a functional prototype that addressed sustainability challenges, while I gained experience in agile teamwork, project management, and balancing both technical and leadership responsibilities.
